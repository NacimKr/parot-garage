# parot-garage

## Clonez le projet via la commande 
````
git clone https://github.com/NacimKr/parot-garage.git
````
### Puis entrez la commande 
````
composer install
````
 ou 
```` 
composer i
````
### Puis lancez le serveur avec la commande 
````
symfony serve
````
